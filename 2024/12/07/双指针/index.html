双指针

一种优化双重for循环的手段, 可将复杂度O(n2)优化为O(n)

一般形式有两种, 快慢指针与相对指针

1: 调整数组顺序使奇数位于偶数前面

思想来源于快速排序, 定义对向指针i, j

i从前往后遍历找偶数, j从后往前遍历找基数, 限制i < j

2: 三值字符串

每次开一个a[10]={0};数组, 存的是各个数字出现的个数

慢指针移动条件 j < i && a[1] && a[2] && a[3]

每次慢指针移动使区间剔除尾部数字, 直到最后一次剔除时则为此次新添加数字满足的的最小长度进行记录

如果条件一次都没走就是没有符合的子串

3: 最长连续不重复子序列

思想: 我们从头到尾遍历元素作为右边界, 我们来维护最远的左边界, 每向区间中加入一个数就有可能重复, 如果重复那么重复的数唯一只能是新加入的数, 这时将左边界向后移动直到没有重复元素, 这时即为区间最长的左边界

那么移动左边界的条件就是右边界的元素是否重复

4: 三值字符串

思想: 同理我们维护一个区间, 开一个数组来记录1,2,3每个数字在区间中出现的个数, 每次右边界后移的时候若使1,2,3的个数都大于0, 那么将左边界要后移至1,2,3任意一个数等于0, 这个过程中计算最短区间

慢指针移动的条件就是1,2,3不为0

5: 判断子序列

思路: 遍历子序列a, 每次慢指针找到第一个相等的元素, 之后慢指针自增找下一个元素, 如果循环过程中慢指针遍历完了b那么就不是子序列, 否则是

6: 骑车路线

本质: 求原序列连续的非严格递增的子序列

思路: 两个指针i, j, j维护左边界, i从头到尾遍历原序列, 每当当前数小于等于下一个数让序列中的个数自增, 如果当前数大于下一个数, 此时判断区间内元素个数是否 >= 2, 如果是就对res进行更新 res = max(res, a[i] - a[j]), j = i + 1;进行左边界的移动, 继续遍历

要注意: 1. cnt区间元素数的初始值1 3. 每一次当前数大于下一个数时都要进行左边界的变化, 以及cnt的重置
